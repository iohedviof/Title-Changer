# Title Changer

Client-side Fabric mod that customizes the Minecraft window title.

## Platform Support

- Loader: Fabric only
- Mod version: 26.1.x

## Dependencies

- [Fabric API](https://modrinth.com/mod/fabric-api)
- [YACL](https://modrinth.com/mod/yacl)
- [Mod Menu](https://modrinth.com/mod/modmenu) (optional)
